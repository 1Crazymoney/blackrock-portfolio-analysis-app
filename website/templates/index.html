APIs Applied

  • Blackrock

    The Blackrock Alladin API provided with stock data that allowed us to view trends in the data on a daily, weekly, and monthly basis.

  • News

    We used an open-source API that allowed us to view a large collection of articles and sort and search by keywords.

  • Flask

    Our website uses Flask as our server in the development of this website.

  • Python/CSS/JS

    Python was used for all data handling and calculation, as well as the backend streams on this website. JavaScript was used for integration with the CSS/HTML frontend.
